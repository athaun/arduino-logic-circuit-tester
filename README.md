## Arduino Logic Circuit Tester

### Usage example - configure in inputs.h
```c
int input_pins[] = {
  8,  // a - green
  9,  // b - yellow
  10, // c - blue
  11  // d - black
};
int result_pin = 4; // White
// Default analog read pin is A0 - also white
```
